# Context Cards

Quick, copy-paste context snippets for common situations.

---

## How to Use

1. Keep active cards updated below
2. Copy relevant card when starting a session
3. Paste after your boot prompt
4. Add specific request

---

## Card Templates

### Project Card
```
PROJECT: [Name]
STATUS: [Phase]
LAST SESSION: [Platform] on [Date]
CONTEXT: [1-2 sentences]
NEXT STEP: [Current priority]
```

### Handoff Card
```
CONTINUING FROM: [Previous Platform]
SESSION: [Date]
COMPLETED: [What was done]
CONTINUE WITH: [Next steps]
KEY CONTEXT: [Important details]
```

### Task Card
```
TASK: [Description]
PROJECT: [If applicable]
CONSTRAINTS: [Requirements or limits]
PREFERRED OUTPUT: [Format or style]
```
